## Overview

Quickly set up your first project to organize documentation in MyCMO.

## Steps

1. Log in at `https://app.mycmo.com`
2. Click "New Project" on the dashboard
3. Enter a project name and optional description
4. Click "Create Project"
5. Start adding your documentation

Projects help you organize docs for different teams or workflows.
