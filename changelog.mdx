## Upgrading MyCMO

Follow these steps to upgrade to the latest version and apply any necessary changes.

<Steps>
  <Step title="Backup Your Data" icon="database">
    Export all projects to avoid data loss.
  </Step>
  <Step title="Update Package" icon="download">
    <CodeGroup tabs="npm,yarn">
    ````bash
    npm install mycmo@latest
    ````
    ````bash
    yarn add mycmo@latest
    ````
    </CodeGroup>
  </Step>
  <Step title="Migrate Config" icon="settings">
    Update your config file if using v1.0.0 changes:
    
    ````json
    {
      "apiEndpoint": "https://api.example.com",
      "apiKey": "YOUR_API_KEY"
    }
    ````
  </Step>
  <Step title="Restart Service" icon="refresh-cw">
    Restart MyCMO to apply updates.
  </Step>
</Steps>

<Callout kind="tip">
After upgrading, review the [release notes](https://dashboard.example.com/releases) for migration guides specific to your setup.
</Callout>
